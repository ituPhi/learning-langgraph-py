# Learning LangGraph

This repository contains two LangGraph learning projects:

## Projects

### 1. introduction-review
Introduction to LangGraph concepts and basic implementations.

**Location:** `introduction-review/`

**Run:**
```bash
cd introduction-review
uv run langgraph dev
```

### 2. ambiant_agents_py
Ambient agents implementation with LangGraph.

**Location:** `ambiant_agents_py/`

**Run:**
```bash
cd ambiant_agents_py
uv run langgraph dev
```

## Setup

Each project has its own dependencies managed by `uv`. Navigate to the project directory and run:

```bash
uv sync
```

## Requirements

- Python 3.12+
- uv package manager
